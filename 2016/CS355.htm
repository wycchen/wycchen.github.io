Summary

Provide the students with the notion of computer and network security. We mainly focus on network security. In particular, this course will include cryptography, cryptoanalysis, and a variety of network attacks and defenses. If time permits, more about recently developed attacks and defenses on computer systems will also be included.

Tentative List of Topics

  1. Overview
  2. One-time pad
  3. Advanced Encryption Standard
  4. Block cipher modes
  5. Diffie-Hellman key exchange protocol
  6. Buffer-overflow attacks
  7. RSA cryptosystem
  8. Cryptographic hash functions
  9. Message authentication code
  10. Digital signatures
  11. Pretty Good Privacy
  12. IP security

Readings

The reference book for the course is: W. Stallings, An Cryptography and Network Security: Principles and Practices, 6/E.
