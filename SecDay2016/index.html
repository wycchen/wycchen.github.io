Talk Abstracts.

Raylin Tso 左瑞麟

Title: Blockchain Technology and Its Applications

Abstract: Fintech is a factor of the fourth industrial revolution that use innovative technology to make financial services more efficient and reliable. Especially, the blockchain technology is considered as a promising candidate within the fintech technologies. A blockchain is a public and decentralized ledger system which records and validate each and every transaction that has ever been executed. It can be coordinated via peer-to-peer (P2P) networks which makes the transaction secure and reliable. In this talk, we will first introduce the history of blockchain by introducing Bitcoin - the decentralized cryptocurrency which uses blockchain as its underlying technology. We will show how it works at a technical level in order to really understand what is special about Bitcoin and blockchain. We will then introduce the potential applications of blockchain. The talk concludes with suggestions for some future researches.

Tsung-Hsuan Hung 洪琮眩

Title: Introduction to Garbling and its Application

Abstract: With the development of cloud computing, how to securely outsource computation to the (untrusted) cloud server becomes a significant security issue. Garbling plays an important role in solving this issue, and in particular serves as a fundamental tool in cryptography and security. In addition, it provides various cryptographic applications and extensions. The notion of garbling is to make the original program P and input x unreadable by generating the corresponding garbled program and input. The garbled program and input can be outsourced to an untrusted party, e.g. cloud server, who can only obtain the output P(x), but cannot extract any significant information about P and x. One of the applications of garbling is secure two-party computation, which allows two parties to jointly compute an arbitrary function on their own inputs without sharing their inputs to the other.

Chia-Mu Yu 游家牧

Title: Deduplicated Cloud Storage Security

Abstract: Cloud storage such as Dropbox and Bitcasa is one of the most popular cloud services. Currently, with the prevalence of mobile cloud computing, users can even collaboratively edit the newest version of documents and synchronize the newest files on their smart mobile devices. A remarkable feature of the current cloud storages is their virtually infinite storage. To support the unlimited storage, the cloud storage provider uses data deduplication technique to reduce the data to be stored and therefore reduce the storage expense. Moreover, the use of data deduplication also helps significantly reduce the need of bandwidth and therefore improve the user experience. Nevertheless, in spite of the above benefits, the data deduplication has its inherent security weaknesses. For example, the adversary may have an unauthorized file downloading via the file hash only. In this talk, we will introduce the inherent security risks of data deduplication, review previous solutions, identify their performance weaknesses, and raise potential countermeasures.
